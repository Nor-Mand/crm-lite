# CRM Lite
CRM lite, is a short application create with Django using Postgresql as Database, for the fronted has Tailwind CSS


## DJANGO - POSTGRESQL - TAILWIND
### Pre-Requisites
`django 4.1.3`
`django-tailwind 3.4.0`
